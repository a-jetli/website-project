Hi, I'm Ansh!

I'm a Computer Science student and I build LLM systems.

I'm a senior CS student at Rutgers University, graduating December 2026, based in central New Jersey. My coursework runs from data structures and algorithms through programming language theory. I've studied languages like Java, C and OCaml, but most of my time now goes into applied AI and backend work in Python.

Most recently, I built an agentic RAG pipeline over ~500 arXiv research papers. It searches with a mix of keyword and semantic retrieval, reranks what it finds, and runs a LangGraph agent that breaks a question down, checks whether it actually has enough to answer, and tries again when it doesn't. It's deployed and you can query it right now. Before that, I made a text RPG engine where an LLM plays dungeon master but a deterministic Python engine owns all the game state and only applies a change after it passes validation. It's like playing through your own adventure book!

Beyond those, I like building things and I care about understanding systems all the way down. I'd love a role where what I built genuinely helps people and solves real problems.
